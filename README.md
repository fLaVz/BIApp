# BIApp

## How to

Placez vous à la racine du projet.  
Lancer l'application en utilisant la commande:  
> python run.py  

Si l'application ne se lance pas, verifiez que vous avez bien toutes les librairies installées:  
* sklearn
* mathplotlib
* numpy
* pandas
* termcolor
* configparser


Tout est affiché en console.
